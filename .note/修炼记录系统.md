# 修炼记录系统

## 架构
- **后端**: Node.js + Express + sql.js (纯JS SQLite)
- **前端**: 原生 HTML/CSS/JS 单页应用
- **认证**: JWT (30天有效期)
- **数据库**: SQLite 文件存储 (`xiulian.db`)

## 关键文件
- `server.js` - Express 服务器，所有 API 路由
- `db.js` - sql.js 数据库初始化，prepare() 辅助函数
- `public/index.html` - SPA 主页面
- `public/style.css` - 暗色玻璃态主题
- `public/app.js` - 前端逻辑

## 评分逻辑（重要）
- **打坐分(50分)**：≥30分钟满分，不足按比例
- **起床分(25分)**：≤9:00满分，每超20分钟扣剩余30%（含不满20分钟的按比例）
- **睡觉分(25分)**：≤0:30满分，每超20分钟扣剩余30%
- **总分 = 打坐分 + 起床分 + 睡觉分**

## 注意事项
- 使用 `sql.js` 替代 `better-sqlite3`，因为后者需要 Visual Studio C++ 编译
- sql.js 的 prepare() 是同步封装，但初始化 getDb() 是异步的
- 数据库每次写入后自动调用 saveDb() 保存到文件
- 日期使用 Asia/Shanghai 时区（前端 `new Date()` 取本地时间）
- 睡觉时间判定：18:00~0:30 为正常，0:30~6:00 为超时
